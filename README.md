# Rapformer 🎧

A custom 14M parameter Autoregressive Transformer language model trained from scratch to generate rap lyrics.

## Tech Stack
* **Engine:** PyTorch
* **Tokenizer:** SentencePiece
* **Backend:** FastAPI & Uvicorn
* **Deployment:** Docker / Hugging Face Spaces
